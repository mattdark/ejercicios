### Escribir un programa que pida tres números y determine el mayor de estos.

```python
# Mayor de tres numeros
a = int(input(“Introduce un numero: ”))
b = int(input(“Introduce un numero: ”))
c = int(input(“Introduce un numero: ”))
if a > b:
    if a > c:
        print(a, “ es el mayor”)
    else:
        print(c, “es el mayor”)
elif b > c:
    print(b, “es el mayor”)
else:
    print(c, “es el mayor”)
```
